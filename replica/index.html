Connect Your Firebase Backend

Enables real auth + Firestore database tracking
1
Create a Firebase project
Go to console.firebase.google.com → Add project → name it replica-app
2
Enable Email/Password Auth
Build → Authentication → Get started → Enable Email/Password
3
Create Firestore Database
Build → Firestore → Create database → Start in test mode
4
Get your config
⚙️ Project Settings → Your apps → Web app → Register app → copy firebaseConfig
5
Set Firestore Rules (important)
Firestore → Rules → replace with:
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /users/{uid} {
      allow read: if request.auth != null;
      allow write: if request.auth.uid == uid;
    }
    match /capsules/{id} {
      allow read: if true;
      allow write: if request.auth != null;
    }
    match /runs/{id} { allow read,write: if request.auth != null; }
    match /llm_logs/{id} { allow read,write: if request.auth != null; }
  }
}
Demo mode = in-memory only. Nothing saved across reloads.
